### MONTH -1: PRODUCTION

#### Week 1: Layout Construction
- [ ] Create new document from master template
- [ ] Apply master pages to appropriate spreads
- [ ] Flow main story text into frames
- [ ] Format headlines and bylines
- [ ] Apply paragraph styles throughout

#### Week 2: Visual Integration
- [ ] Place cover illustration
- [ ] Import interior artwork
- [ ] Adjust text wrap around images
- [ ] Add pull quotes and sidebars
- [ ] Insert ornamental dividers
- [ ] Fine-tune spacing and alignment

#### Week 3: Refinement
- [ ] Complete all remaining pages
- [ ] Add page numbers
- [ ] Insert headers/footers
- [ ] Proof all text (spell-check, formatting)
- [ ] Check image resolution (300 DPI minimum)
- [ ] Verify bleed extends properly
- [ ] Review color usage (CMYK compliance)

#### Week 4: Pre-Press
- [ ] Run preflight check (fix all errors)
- [ ] Export PDF/X-1a for printing
- [ ] Review PDF carefully (every page)
- [ ] Create backup/archive of working file
- [ ] Send test file to printer for quote/approval
- [ ] Make any requested printer adjustments
- [ ] Finalize print count
- [ ] Submit final files to printer

## DISTRIBUTION CHANNELS

### Direct Sales (Highest Margin)
- **Website/Online Store:** Keep 100% after payment processing (~3%)
- **Convention Sales:** Face-to-face, immediate payment
- **Pre-orders:** Reduce risk, confirm demand

### Consignment (Medium Margin)
- **Comic Shops:** 60/40 or 50/50 split typical
- **Independent Bookstores:** Similar terms
- **Zine Distros:** 50/50 usually
- **Cafes/Galleries:** Varies, often 70/30 in your favor

### Subscription Service (Best Long-Term)
- **Direct Subscriptions:** Annual, consistent revenue
- **Patreon:** Monthly support, bonus content
- **Bundle Deals:** Discount for commitment

## RISK MITIGATION

### Common Pitfalls & Solutions

**Problem:** Content delays  
**Solution:** Build 2-week buffer, maintain backup content

**Problem:** Budget overruns  
**Solution:** Start small (print 100 first run), scale with demand

**Problem:** Low initial sales  
**Solution:** Focus on quality over quantity, build slowly

**Problem:** Contributor inconsistency  
**Solution:** Develop reliable core team, have multiple backups

**Problem:** Technical printing issues  
**Solution:** Always order printer proof, build relationships with printers

**Problem:** Burnout  
**Solution:** Maintain realistic schedule, take breaks between volumes

## PIVOT POINTS

### When to Adjust

**If sales are lower than expected:**
- Reduce print quantities
- Focus on direct sales over consignment
- Increase marketing efforts
- Consider digital-only as supplement
- Don't give up too quickly (build takes time)

**If costs are too high:**
- Negotiate with contributors (profit-share model)
- Reduce page count slightly
- Find cheaper printing options
- Simplify design (less color, simpler layouts)
- Consider black + one color instead of full CMYK

**If production is too demanding:**
- Move to bi-monthly schedule
- Reduce page count
- Simplify certain sections
- Build stronger team
- Automate where possible

## FINAL PRE-FLIGHT CHECKLIST

### Before Sending ANYTHING to Printer:

- [ ] Every page proofread by at least 2 people
- [ ] All images 300 DPI or higher
- [ ] All colors in CMYK (no RGB)
- [ ] All fonts embedded
- [ ] Bleed extends 0.125" on all sides
- [ ] No important content within 0.25" of trim
- [ ] Preflight shows zero errors
- [ ] PDF exported with correct settings
- [ ] File tested in PDF reader
- [ ] Page count correct (must be multiple of 4)
- [ ] Spreads arranged correctly for folding
- [ ] Cover orientation correct
- [ ] File name clear and dated
- [ ] Backup created
- [ ] Sleep on it, review again tomorrow
