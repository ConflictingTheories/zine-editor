## DETAILED ASSET SPECIFICATIONS

### 1. CHARACTER SILHOUETTE TEMPLATES

#### Template A: Standing Hero Pose
```
Dimensions: 3" height, proportional width
Format: SVG vector
Color: Solid black (#0A0A0A)
Details:
- Strong stance, feet shoulder-width
- Cape flowing (if applicable)
- One hand on weapon or hip
- Clear head/shoulder silhouette
- No internal details, pure shape

File Naming: char_hero_standing_01.svg
```

#### Template B: Action Pose (Dynamic)
```
Dimensions: 4" diagonal
Format: SVG vector
Color: Solid black or accent color
Details:
- Mid-action (punch, kick, spell-cast)
- Exaggerated perspective
- Flowing hair/clothing for motion
- Weight shifted dramatically
- Strong diagonal composition

File Naming: char_hero_action_01.svg
```

#### Template C: Profile/Iconic Pose
```
Dimensions: 2" height (smaller, accent use)
Format: SVG vector
Color: Solid black
Details:
- Side profile, distinctive silhouette
- Recognizable by outline alone
- Minimal but clear details
- Good for page corners/margins

File Naming: char_hero_profile_01.svg
```

#### Template D: Monster/Demon Silhouette
```
Dimensions: Variable, typically 3-5" height
Format: SVG vector
Color: Black or Blood Crimson
Details:
- Threatening pose
- Exaggerated features (horns, claws, wings)
- Non-human proportions okay
- High contrast shapes
- Can be more complex than human silhouettes

File Naming: creature_demon_01.svg
```

### 2. PANEL FRAME SPECIFICATIONS

#### Standard Rectangle Set
```
Create 5 sizes:

Small: 1.5" × 1.5" (square, reaction shots)
Medium Portrait: 2" × 3" (vertical)
Medium Landscape: 3" × 2" (horizontal)
Large: 4" × 3" (main action)
Full Width: 5" × 2.5" (establishing shot)

All Frames:
- Stroke: 4pt solid black
- No fill (transparent center)
- Corner radius: 0 (sharp) or 0.0625" (slightly rounded)
- Corners: mitered

File Naming: panel_rect_[size]_[number].svg
Example: panel_rect_large_01.svg
```

#### Dynamic Frame Template
```
Purpose: Action, chaos, intensity

Shape: Irregular polygon (5-8 sides)
Stroke: 6pt black with rough/grunge edge
Angles: Sharp, aggressive (no curves)
Size: Variable, typically 3-4" across

Creation Method:
1. Start with triangle or pentagon
2. Add/remove vertices for complexity
3. Rotate 15-30° for dynamism
4. Apply roughen effect for edge variation

File Naming: panel_dynamic_01.svg
```

#### Medieval Ornate Frame
```
Purpose: Flashbacks, historical scenes, magical moments

Base Shape: Rectangle or arch
Decoration: Corner flourishes
Style: Celtic knot, Gothic tracery, or Art Nouveau
Stroke: 3pt black for frame, 2pt for decoration
Size: 3" × 4" typical

Elements:
- Corner pieces (separate assets, can be reused)
- Optional: Top arch or decorative header
- Edge pattern: repeating medieval motif

File Naming: panel_medieval_01.svg
File Naming (corners): ornament_corner_medieval_01.svg
```

#### Tech/Glitch Frame
```
Purpose: Digital scenes, AI communication, cyber environments

Base Shape: Rectangle
Style: Digital corruption, scan lines, pixel glitch
Stroke: 4pt with breaks/gaps
Additional: Corner brackets, data readout text

Effects:
- Intentional gaps in border
- Pixelated segments
- "Loading" or "Error" indicators
- Color: Eldritch Green or Lightning Blue option

File Naming: panel_tech_01.svg
```

### 3. SPEECH BALLOON LIBRARY

#### Standard Dialog Balloon
```
Shape: Horizontal ellipse
Dimensions: 2" width × 1.25" height (minimum)
Stroke: 2pt black, solid
Fill: White (#FFFFFF)
Tail: Curved, pointing to speaker
Tail length: 0.5"

Text Area:
- Inner padding: 0.125" all sides
- Max text size: 10pt
- Alignment: Center or left

File Naming: balloon_dialog_standard.svg
```

#### Shout Balloon (Explosive)
```
Shape: Star-burst, 8-12 points
Dimensions: 2.5" across
Stroke: 4pt black, bold angular segments
Fill: White
Tail: Sharp, angular, bold

Optional:
- Radiating lines around balloon (motion)
- Slight perspective skew for 3D effect

File Naming: balloon_shout_01.svg
```

#### Thought Balloon (Cloud)
```
Shape: Fluffy cloud (3-5 connected circles)
Dimensions: 2" across
Stroke: 2pt black
Fill: White with 90% opacity
Tail: 3 diminishing circles connecting to thinker

Variation: Can have dotted stroke instead of solid

File Naming: balloon_thought_01.svg
```

#### Caption Box (Narrator)
```
Shape: Rectangle
Dimensions: 4" width × 0.75" height
Stroke: None or 1pt border
Fill: Solid black or themed color
Text: White or contrasting, 9pt

Variations:
- TOP: For scene-setting captions
- BOTTOM: For narrative conclusion
- SIDE: For editorial commentary

File Naming: caption_box_narrator.svg
```

#### Whisper Balloon
```
Shape: Small ellipse
Dimensions: 1.5" × 0.75"
Stroke: 1.5pt dashed line, black
Fill: White
Tail: Thin, delicate curve

File Naming: balloon_whisper.svg
```

#### Radio/Phone Balloon
```
Shape: Rectangle with antenna/signal icon
Dimensions: 2.5" × 1.5"
Stroke: 2pt solid
Fill: Black
Text: Eldritch Green, monospace font
Decoration: Signal waves, circuit pattern on border
```

#### Demonic Speech Balloon
```

Stroke: 3pt with distortion/roughen
```

## BATCH CREATION WORKFLOWS

### Workflow 1: Creating 10 Character Silhouettes (One Session)

**Preparation:**
- [ ] Collect reference images (poses, characters)
- [ ] Set up Affinity Designer template (3" × 3" artboards)
- [ ] Prepare export settings

**Production (20 minutes per silhouette):**
1. Import reference (low opacity layer)
2. Pen Tool: Trace bold outline
3. Simplify complex areas
4. Close all paths
5. Fill with solid black
6. Delete reference layer
7. Export SVG
8. Repeat for next character

**Post-Production:**
- [ ] Import all to Asset Library in Publisher
- [ ] Test scale and placement
- [ ] Organize in "Character Silhouettes" category

### Workflow 2: Speech Balloon Set (90 minutes)

**Create these variations:**
1. Standard dialog (×3 sizes)
2. Shout (×2 styles)
3. Thought (×2 styles)
4. Whisper
5. Caption box (×3 positions)
6. Tech/radio
7. Demonic

**Per Balloon (7 minutes average):**
1. Create shape (ellipse, rectangle, custom)
2. Add tail with Pen Tool
3. Boolean combine
4. Set stroke and fill
5. Export SVG
6. Add to Asset Library

### Workflow 3: SFX Text Library (2 hours)

**Priority SFX (create these first):**
- CRASH, BOOM, BANG, SLASH, ZAP, WHOOSH
- CRACK, THWAM, BZZT, FWOOSH
- 10 total = critical mass

**Per SFX (10-12 minutes):**
1. Type text (bold, large)
2. Artistic Text Tool: Adjust letters
3. Add outline stroke (2-3pt)
4. Optional: 3D extrusion or shadow
5. Convert text to curves
6. Export SVG
7. Add to Library

### Workflow 4: Pattern Library (3 hours)

**Essential Patterns:**
- 3 halftone screens
- 2 tech patterns (circuit, grid)
- 3 textures (stone, parchment, metal)
- 2 distortion effects

**Per Pattern (15-20 minutes):**
1. Create design in 4" × 4" area
2. Test seamless tiling
3. Export PNG (300 DPI) or SVG
4. In Publisher: Import, set as pattern fill
5. Save as Swatch or Object Style
6. Catalog in Asset Library

## ADVANCED: CREATING VOLUME-SPECIFIC SIGIL

Each volume needs a unique identifying symbol:

**Process:**
1. **Concept:** Relate to volume's theme
   - Vol I (Awakening): Opening eye
   - Vol II (Descent): Downward arrow/spiral
   - Vol III (Betrayal): Broken sword/fractured heart

2. **Design:**
   - Geometric base (circle, hexagon, triangle)
   - Central motif
   - Surrounding details
   - 2-3 colors maximum

3. **Application:**
   - Cover: Prominent (2-3" size)
   - Interior: Watermark (10% opacity)
   - Page headers: Small (0.5" size)
   - Spine: If visible

4. **Creation in Designer:**
   - Start with geometric grid
   - Build symbol symmetrically
   - Clean, bold lines
   - Export multiple sizes

**File Naming:** sigil_vol[number]_[theme].svg
**Example:** sigil_vol01_awakening.svg

## CONTRIBUTOR GUIDELINES

If working with artists:

**Asset Submission Requirements:**
- Format: SVG (vector) or PNG 300+ DPI
- Color: Black (#0A0A0A) unless specified
- Size: Minimum 3" largest dimension
- Clean edges: No background, transparent
- Naming: Follow established convention
- Delivery: Organized folder, labeled clearly

**Rights & Attribution:**
- Clarify usage rights
- Discuss attribution (credit page)
- Agree on payment/compensation
- Set delivery timeline
- Establish revision policy
