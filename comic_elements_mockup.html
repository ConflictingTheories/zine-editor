Implementation Examples

Character Silhouettes

Usage: Layer over backgrounds to create depth, establish characters, or show action without detailed illustration.

Blend Mode: Multiply for dark backgrounds, Screen for light.

Opacity: 100% for main subjects, 30-50% for background figures.

Pro Tip: Break panel boundaries with silhouettes to show dimensional bleeding.

Speech Balloons

Standard Dialog: Round or elliptical, clean tail pointing to speaker.

Shouting: Jagged, explosive outline with bold border (4pt+).

Thought: Cloud shape with diminishing circle tail.

Demonic/Magic: Inverted colors, distorted edges, custom fonts.

SFX Integration

Action: Large, bold, integrated with motion (CRASH! BOOM! SLASH!)

Ambient: Smaller, subtle, atmospheric (hummm... drip... whisper...)

Tech/Magic: Include symbols, glowing effects, themed colors.

Font: Comic-style display fonts (Bangers, Blambot fonts).

Motion Effects

Speed Lines: Radiate from action source, tapered, 50-70% opacity.

Impact Bursts: Star pattern from collision point.

Energy Trails: Flowing curves with particles, themed colors.

Placement: Separate layer, behind characters, adjustable blend mode.

Panel Frames

Standard: 4pt black border, clean rectangles.

Dynamic: Angled, irregular shapes for action/chaos.

Themed: Ornate medieval or tech-styled borders.

Gutter: 0.25" standard, can vary for pacing.

Overlay Patterns

Halftone: 10-20% opacity, vintage comic feel.

Tech Patterns: Circuits, grids for digital scenes.

Medieval: Stone, parchment for historical.

Application: Apply to background layer, subtle enhancement.

Layering Workflow

Layer Stack (Bottom to Top):

  1. Background Patterns — Halftones, textures at 10-20% opacity
  2. Panel Frames — Borders defining comic spaces
  3. Main Images — Photos, illustrations, backgrounds
  4. Character Silhouettes — Layered figures, blend mode adjusted
  5. Motion Effects — Speed lines, impact bursts
  6. Text Frames — Body text, captions
  7. Speech Balloons — Dialog, thoughts with integrated text
  8. SFX Text — Sound effects overlays
  9. Foreground Elements — Frame-breaking elements

"Layer reality. Each element adds depth to the narrative dimensional collapse."
